# 分时图
## 1. 开盘急拉别追买, 急拉一般有回踩
## 2. 买时需看右盘口, 压盘出现快入手
## 3. 开盘急跌别追卖, 急跌一般有反弹
## 4. 卖时也看右盘口, 托盘出现快脱手()

## 5.卖盘很多,  买盘很少,  (压盘) __跟风盘 买入时机
> 1.控制股价上涨速度, 在较低价位能吃到更多的筹码
> 2.恐吓散户, 让散户以为抛压大不敢买入, 甚至交出筹码
> 3.让散户保留足够的资金, 能在后期高位接盘
## 6.卖盘很少,  买盘很多   (托盘) __  出手时机
> 1.控制股价下跌速度, 在较高位置卖出筹码
> 2.给散户信心, 觉得股价不会跌, 而买入接盘
> 3.让散户保留足够的筹码, 能在后期割肉卖出给主力
## 7.买卖盘比较均衡, 可以出手(时机)
